# breakout-rooms

breakout roomsは、オンライン会議のディスカッションを簡単にいくつかのルームに分類できるアプリケーションです。

# usage

addでユーザを追加して、1部屋当たりの人数を選んだらAssignを押すだけ。

# Known Issues

人数 / 一部屋当たりの人数を10 / 4になるように指定すると、4人のルームと6人のルームが出来上がるオシャレ~~（残念）~~仕様です・・・

# Installation
- npm install
- npm run start
